## 실행 절차

1. 위 6개 DB에서 `notion-query-database-view`로 각 팀원의 **가장 최신 보고서** 1건씩 조회한다 (시작일 기준 최신순 첫 번째 항목).
2. 각 보고서 URL로 `notion-fetch`를 실행해 페이지 본문 내용을 가져온다.
3. 아래 형식으로 관리자용 요약을 작성한다:

---

**요약 형식:**

```
## 제출 현황
(팀원별 주차 / 상태 표 — 미제출자 별도 표시)

## [이름] — [담당 업무 키워드]
(핵심 내용 3~5줄. 수치가 있으면 수치 포함.)

(6명 반복)

## 관리자 확인 사항
(이슈, 리스크, 미제출, 중요 일정 등을 표로 정리)
```

---

4. 작성한 요약을 `notion-create-pages`로 **노션 개인 페이지(parent 생략)**에 저장한다.
   - 페이지 제목: `영업본부 주간보고 요약 (YYYY-MM-DD ~ MM-DD)`
   - 아이콘: 📋

5. 저장 완료 후 노션 페이지 URL을 사용자에게 출력한다.
